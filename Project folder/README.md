# Mini CBT Flask App

This project is a simple Computer Based Test (CBT) system built with Flask.

Features
- Displays multiple questions
- Tracks the user's score
- Uses Object Oriented Programming
- Uses a Stack data structure
- Records submission time using datetime

How to Run

1. Install Flask

pip install flask

2. Run the application

python app.py

3. Open browser and go to

http://127.0.0.1:5000

You can now take the CBT test
